# 📋 תיקיית קונפיגורציה

תיקייה זו מכילה קבצי קונפיגורציה חיצוניים המאפשרים התאמה והרחבה של המערכת **ללא שינוי בקוד המקור**.

## מטרות

* התאמות ארגוניות או ניסיוניות
* שינוי התנהגות מערכתית
* שמירה על מפרט הפרויקט כ-Single Source of Vision

## סדר עדיפות טעינה

1. `index.html` – Bootstrap עם `window.__INITIAL_FLOW_DATA__`
2. JSON ברירת מחדל מ-`src/protocols/`
3. קבצי קונפיגורציה מתיקייה זו
4. Runtime State

## קבצים אפשריים

* `feature-flags.json` – דגלים להפעלה/כיבוי פיצ'רים
* `flow-overrides.json` – עקיפת זרימות קיימות
* `ui-behavior.json` – התאמות UI
* `environment.json` – משתני סביבה

## כללים

* קונפיגורציה **משנה** התנהגות, לא מוחקת דאטא
* קונפיגורציה **גוברת** על ברירת מחדל
* אין לשמור PII או סודות בתיקייה זו
