# מקור אמת לקורס BLS - איחוד הצלה

## מטרה

התיקייה הזו מרכזת שכבת `source of truth` מסודרת עבור חומרי הקורס מתוך תיקיית Google Drive בשם `איחוד הצלה`.

המטרה בשלב הזה:

1. לשמר את מבנה הקורס המקורי.
2. לקשר כל יחידת ידע לקובץ המקורי בדרייב.
3. להחזיק עותק טקסטואלי מסודר, לפני כל התאמה לתרשים הזרימה של האתר.
4. ליצור בסיס עבודה בטוח לאימות, תיקון והרחבת התוכן באתר.

## עקרונות עבודה

- תיקיית הדרייב היא מקור האמת הראשי.
- הקבצים כאן הם שכבת ארגון והנגשה, לא מקור חלופי.
- אין לבצע כאן "שכתוב קליני" בלי לציין זאת במפורש.
- כל מסמך חייב לכלול:
  - שם הקובץ המקורי
  - מזהה או קישור ל-Drive
  - תאריך שליפה
  - סטטוס עיבוד
- בשלב זה נשמרת עדיפות לנאמנות למקור על פני ניקיון ניסוחי.

## מבנה התיקייה

- [course-inventory.md](/Users/evyatarhazan/Desktop/project/emergency-protocol-diagram/docs/source-truth/united-hatzalah-bls/course-inventory.md) - מפת כל הקבצים שנמצאו בתיקיית הדרייב.
- `modules/` - קבצי מקור מאורגנים לפי יחידות הקורס.

## סטטוס נוכחי

- אינדקס מלא של תיקיית הדרייב נבנה.
- הושלמה קליטה ראשונית ליחידות הליבה הרלוונטיות ביותר ל-BLS:
  - מבוא לקורס
  - מבוא להחייאה
  - החייאת מבוגר מטפל יחיד
  - החייאה בזוג ומכשירים
  - החייאת ילדים, תינוקות וחנק
  - אנמנזה וגישה לחולה
- יתר חומרי הקורס עדיין ממתינים לקליטה באותה מתכונת.

## שימוש עתידי

לאחר שכלל חומרי הקורס ייקלטו:

1. נמפה כל יחידת קורס למסכי האתר.
2. נזהה פערים בין מבנה הקורס לבין מבנה הפרוטוקול.
3. נבנה שכבת אימות קליני עם מקורות, הערות, ופערים.
4. נסמן במפורש מה מתאים ל-BLS, מה מתאים לחובש/נהג אמבולנס, ומה דורש ALS.
